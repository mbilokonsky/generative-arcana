# Majors — Borrowed

Map the 22 majors 1:1 onto an existing 22-element structure the theme already contains — 22 named figures, 22 stations of a path, 22 letters, 22 chapters, 22 trumps of an in-world deck.

No major-rank layer — the 22 cards are generated directly, integrated per `references/integration.md` with `major_arcana.visual_style` (declared) and each slot's `station_slug` (sublimated — fixed by the major walk).

## Procedure

1. **Identify the source set.** Name the structure and confirm its count. If 20–24, decide deliberately how to reach exactly 22 (merge near-duplicates, split a compound, add/drop with a rationale). Far from 22 → use `journey.md` or `primes.md`.

2. **Order it.** Adopt the source's canonical order as 0→21; sanity-check that position 0 reads as precondition/potential and 21 as completion. Reorder only if the source order yields an incoherent arc.

3. **Map each element to a card.** Define name + meaning palette + depicted content from the element's own associations. Render each *as a tarot card* — give it upright and chiral-inverted senses (`references/integration.md`) even if the source presents it one-sidedly.

4. **Reconcile with the fixed station and number.** Each slot's station is determined by the major walk independently of the borrowed order. Where the borrowed element and its station *resonate*, lean in (the element may wear the station at surface). Where they *clash*, accept the productive tension and keep the station sublimated (it tints rather than fights). The slot's prime/composite character is also derivable and latent — note where a prime-positioned element happens to be an irreducible one in the source (a nice reinforcement), but don't force it.

5. **Build the cards** — integrate each per `references/integration.md`.

## Quality

- All 22 source elements present, none padded or doubled.
- Each reads as a card (two-sided meaning), not a glossary entry.
- A coherent precondition→completion arc; station resonance leaned into or sublimated where it would clash.
